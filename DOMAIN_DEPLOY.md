# Инструкция по деплою на домен

## Варианты деплоя на домен

### Вариант 1: Бесплатный хостинг с подключением домена (Рекомендуется)

#### Netlify (самый простой способ)

1. **Зарегистрируйтесь на [netlify.com](https://netlify.com)**

2. **Загрузите проект:**
   - Перетащите всю папку проекта в Netlify Drop
   - Или подключите GitHub репозиторий

3. **Настройки деплоя:**
   - Build command: (оставить пустым)
   - Publish directory: `.` (точка - корневая папка)

4. **Подключение домена:**
   - Перейдите в Domain settings
   - Нажмите "Add custom domain"
   - Введите ваш домен (например: blackwood.uz)
   - Следуйте инструкциям по настройке DNS

5. **Настройка DNS у регистратора домена:**
   ```
   Тип: A
   Имя: @
   Значение: 75.2.60.5
   
   Тип: CNAME
   Имя: www
   Значение: ваш-сайт.netlify.app
   ```

#### Vercel (альтернатива)

1. Зарегистрируйтесь на [vercel.com](https://vercel.com)
2. Импортируйте проект
3. Добавьте домен в настройках
4. Настройте DNS записи (A и CNAME)

### Вариант 2: Обычный хостинг (cPanel, FTP)

Если у вас уже есть хостинг с доменом:

1. **Подготовка файлов:**
   - Убедитесь, что все файлы готовы
   - Файл `.htaccess` уже создан

2. **Загрузка через FTP:**
   - Используйте FTP клиент (FileZilla, WinSCP)
   - Подключитесь к серверу
   - Загрузите все файлы в папку `public_html` или `www`

3. **Структура на сервере:**
   ```
   public_html/
   ├── index.html
   ├── style.css
   ├── .htaccess
   ├── robots.txt
   ├── sitemap.xml
   ├── Images/
   └── icons/
   ```

4. **Проверка:**
   - Откройте ваш домен в браузере
   - Убедитесь, что все работает

### Вариант 3: VPS/Сервер (для продвинутых)

Если у вас есть VPS сервер:

1. **Установите веб-сервер:**
   ```bash
   # Ubuntu/Debian
   sudo apt update
   sudo apt install nginx
   ```

2. **Загрузите файлы:**
   ```bash
   # Создайте директорию
   sudo mkdir -p /var/www/blackwood.uz
   
   # Загрузите файлы (через SCP, FTP или Git)
   ```

3. **Настройте Nginx:**
   Создайте файл `/etc/nginx/sites-available/blackwood.uz`:
   ```nginx
   server {
       listen 80;
       server_name blackwood.uz www.blackwood.uz;
       root /var/www/blackwood.uz;
       index index.html;

       location / {
           try_files $uri $uri/ /index.html;
       }

       # Кэширование
       location ~* \.(jpg|jpeg|png|gif|ico|svg|css|js)$ {
           expires 1y;
           add_header Cache-Control "public, immutable";
       }
   }
   ```

4. **Активируйте сайт:**
   ```bash
   sudo ln -s /etc/nginx/sites-available/blackwood.uz /etc/nginx/sites-enabled/
   sudo nginx -t
   sudo systemctl reload nginx
   ```

## Настройка DNS для домена

### Если используете Netlify/Vercel:

1. Зайдите в панель управления доменом (у регистратора)
2. Найдите раздел DNS/Nameservers
3. Добавьте записи:
   - **A запись**: @ → IP адрес (получите в Netlify/Vercel)
   - **CNAME запись**: www → ваш-сайт.netlify.app

### Если используете обычный хостинг:

1. Узнайте IP адрес сервера у хостинг-провайдера
2. Настройте A запись: @ → IP адрес сервера
3. Настройте CNAME: www → ваш-домен.uz

## Проверка после деплоя

1. Откройте ваш домен в браузере
2. Проверьте работу всех страниц
3. Проверьте загрузку изображений
4. Проверьте мобильную версию
5. Проверьте скорость: [PageSpeed Insights](https://pagespeed.web.dev/)

## SSL сертификат (HTTPS)

### Netlify/Vercel:
- SSL сертификат выдается автоматически и бесплатно
- Просто включите HTTPS в настройках

### Обычный хостинг:
- Используйте Let's Encrypt (бесплатно)
- Или купите SSL сертификат у хостинг-провайдера
