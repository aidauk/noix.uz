# Где найти "Set up Netlify DNS" в Netlify - Подробная инструкция

## Пошаговая инструкция с точными местами

### Шаг 1: Добавление домена

1. Зайдите на [netlify.com](https://netlify.com) и войдите в аккаунт
2. Выберите ваш сайт (кликните на название сайта в списке)
3. В верхнем меню сайта нажмите **"Site settings"** (или **"Настройки сайта"**)
4. В левом боковом меню найдите и нажмите **"Domain management"** (или **"Управление доменами"**)
5. Нажмите кнопку **"Add custom domain"** (или **"Добавить домен"**)
6. Введите домен: **noix.uz**
7. Нажмите **"Verify"** (или **"Проверить"**)

### Шаг 2: Где найти "Set up Netlify DNS"

После добавления домена вы увидите несколько вариантов настройки DNS.

**Вариант 1: Если видите кнопку "Set up Netlify DNS"**

1. В разделе **"Domain management"** вы увидите ваш домен **noix.uz**
2. Рядом с доменом будет статус (обычно желтый или оранжевый индикатор)
3. Под доменом или рядом с ним будет кнопка **"Set up Netlify DNS"** или **"Configure Netlify DNS"**
4. Нажмите на эту кнопку

**Вариант 2: Если кнопки нет, используйте DNS регистратора**

Если вы не видите кнопку "Set up Netlify DNS", используйте вариант с DNS записями:

1. В разделе **"Domain management"** найдите ваш домен **noix.uz**
2. Рядом с доменом нажмите на три точки (**...**) или кнопку **"Options"**
3. Выберите **"Check DNS configuration"** или **"Verify DNS"**
4. Netlify покажет вам DNS записи, которые нужно добавить

### Шаг 3: Получение Nameservers (если используете Netlify DNS)

После нажатия "Set up Netlify DNS":

1. Netlify покажет вам **4 Nameservers** (DNS серверы)
2. Они будут выглядеть примерно так:
   ```
   dns1.p01.nsone.net
   dns2.p01.nsone.net
   dns3.p01.nsone.net
   dns4.p01.nsone.net
   ```
3. **Скопируйте эти Nameservers** - они понадобятся для настройки в ahost.uz

### Альтернативный путь (если не видите кнопку)

Если кнопки "Set up Netlify DNS" нет:

1. В **"Domain management"** найдите домен **noix.uz**
2. Нажмите на сам домен (кликните на название)
3. Откроется детальная страница домена
4. Там будет раздел **"DNS configuration"** или **"DNS настройки"**
5. Выберите **"Use Netlify DNS"** или **"Set up Netlify DNS"**

```

│   │   ├── Domain management (левое меню)
│   │   │   ├── Add custom domain (кнопка)
│   │   │   │   └── Ввести: noix.uz

│   │   │   │

│   │   │       ├── Set up Netlify DNS (кнопка) ← ВОТ ОНА!
│   │   │       └── или
│   │   │           └── Check DNS configuration
```

## Если не можете найти

**Альтернативный способ:**

1. В **"Domain management"** найдите домен **noix.uz**
2. Рядом с доменом будет статус (Pending, Ready, или ошибка)
3. Если статус **"Pending"** или показывает ошибку DNS:
   - Нажмите на сам домен (кликните на **noix.uz**)
   - Откроется детальная страница
   - Там будет раздел с DNS настройками
   - Выберите **"Use Netlify DNS"**

---

## Что делать дальше

После нажатия "Set up Netlify DNS":

1. Netlify покажет вам Nameservers
2. Скопируйте их
3. Зайдите на [ahost.uz](https://ahost.uz)
4. Найдите управление DNS для домена noix.uz
5. Замените текущие Nameservers на те, что дал Netlify
6. Сохраните изменения
