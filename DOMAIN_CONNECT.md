# Инструкция по подключению домена

## Вариант 1: Подключение домена к Netlify (Рекомендуется)

### Шаг 1: Деплой сайта на Netlify

1. Зайдите на [netlify.com](https://netlify.com)
2. Войдите в аккаунт (или создайте новый)
3. Нажмите **"Add new site"** → **"Import an existing project"**
4. Выберите **GitHub** и авторизуйтесь
5. Выберите репозиторий: **aidauk/blackwood.uz**
6. Настройки деплоя:
   - **Build command:** (оставить пустым)
   - **Publish directory:** `.` (точка - корневая папка)
7. Нажмите **"Deploy site"**
8. Дождитесь завершения деплоя (обычно 1-2 минуты)

### Шаг 2: Добавление домена в Netlify

1. В Netlify перейдите в ваш сайт
2. Откройте **Site settings** → **Domain management**
3. Нажмите **"Add custom domain"**
4. Введите ваш домен: `blackwood.uz` (или ваш домен)
5. Нажмите **"Verify"**

### Шаг 3: Настройка DNS

Netlify покажет вам два варианта настройки DNS:

#### Вариант A: Использовать Netlify DNS (Проще и рекомендуется)

1. В Netlify нажмите **"Set up Netlify DNS"**
2. Netlify даст вам **Nameservers** (NS записи), например:
   ```
   dns1.p01.nsone.net
   dns2.p01.nsone.net
   dns3.p01.nsone.net
   dns4.p01.nsone.net
   ```

3. Зайдите в панель управления доменом (у регистратора, где покупали домен)
4. Найдите раздел **DNS** или **Nameservers**
5. Замените текущие Nameservers на те, что дал Netlify
6. Сохраните изменения

#### Вариант B: Использовать DNS регистратора

Если хотите оставить DNS у регистратора, добавьте записи:

1. В Netlify нажмите **"Check DNS configuration"**
2. Netlify покажет нужные DNS записи:

   **A запись:**
   ```
   Тип: A
   Имя: @
   Значение: 75.2.60.5
   TTL: 3600
   ```

   **CNAME запись:**
   ```
   Тип: CNAME
   Имя: www
   Значение: ваш-сайт.netlify.app
   TTL: 3600
   ```

3. Зайдите в панель управления доменом
4. Добавьте эти записи в DNS настройки
5. Сохраните изменения

### Шаг 4: Ожидание активации

- DNS изменения могут занять от **5 минут до 48 часов**
- Обычно активируется за **10-30 минут**
- Netlify автоматически выдаст SSL сертификат (HTTPS)

### Шаг 5: Проверка

После активации:
- ✅ `blackwood.uz` будет работать
- ✅ `www.blackwood.uz` будет работать
- ✅ Автоматический HTTPS (SSL)
- ✅ Быстрая загрузка благодаря CDN

## Вариант 2: Подключение домена к обычному хостингу

Если у вас уже есть хостинг и вы загрузили файлы через FTP:

### Шаг 1: Настройка DNS у регистратора

1. Зайдите в панель управления доменом
2. Найдите раздел **DNS** или **DNS Management**
3. Добавьте/измените записи:

   **A запись:**
   ```
   Тип: A
   Имя: @
   Значение: IP адрес вашего хостинга (уточните у хостинг-провайдера)
   TTL: 3600
   ```

   **CNAME запись:**
   ```
   Тип: CNAME
   Имя: www
   Значение: ваш-домен.uz
   TTL: 3600
   ```

4. Сохраните изменения

### Шаг 2: Настройка домена на хостинге

1. Зайдите в панель управления хостингом (cPanel)
2. Найдите **"Addon Domains"** или **"Домены"**
3. Добавьте ваш домен: `blackwood.uz`
4. Укажите папку, где находятся файлы (обычно `public_html`)

### Шаг 3: SSL сертификат

1. В cPanel найдите **"SSL/TLS Status"** или **"Let's Encrypt"**
2. Установите бесплатный SSL сертификат для домена
3. Включите принудительное перенаправление на HTTPS

### Шаг 4: Проверка

Откройте `https://blackwood.uz` в браузере - сайт должен работать!
